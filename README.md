# Embedded Standard Library (esl)

My collection of good to have algorithms, containers, etc, for embedded use.
Active WIP.

## Standard

Designed for C++14 or higher.

## Include

To use this library, add `./src` to your includes and:

```C++
#include <esl/esl.hpp>
```
